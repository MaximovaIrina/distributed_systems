Публичная библиотека

Для сотрудников
     Получить список всех книг, имеющихся в библиотеке 
        Возвращает массив записей,  имеющих следующую структуру:
            1. код ISBN
            2. список авторов (автор представляется фамилией и инициалами)
            3. год издания
            4. издательство
            5. наименование книги
            6. количество экземпляров доступных для читателя
               (если все экземпляры книги на руках, будет равно нулю)
     Добавить новую книгу 
        Принимает:
            1. код ISBNs
            2. список авторов (автор представляется фамилией и инициалами)
            3. год издания
            4. издательство
            5. наименование книги
     Добавить новый экземпляр книги
        Принимает:
            1. код ISBN
            2. количество добавляемых экземпляров

Для читателей
     Получить список всех книг, имеющихся в библиотеке 
        Возвращает массив записей, имеющих следующую структуру:
            1. код ISBN
            2. список авторов (автор представляется фамилией и инициалами)
            3. год издания
            4. издательство
            5. наименование книги
            6. количество экземпляров доступных для читателя 
               (если все экземпляры книги на руках, будет равно нулю)
     Взять книгу 
        Принимает:
            1. код ISBN
        Возвращает:
            1. статус – успешно проведена операция или нет
     Вернуть книгу 
        Принимает: 
            1. код ISBN
	
	
Построение кода сервера на клиенте (очисти \wsclient\src\main\java\ws): 
	cd C:\...\wsclient\src\main\java
	C:\...\wsimport.exe  http://localhost:8080/Library?wsdl -keep
